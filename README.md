🌦️ Simple Weather App using Python and API to fetch real-time temperature and humidity

A simple and user-friendly Weather App built using Python.
This project fetches real-time weather data using an API and displays important details like temperature and humidity.

🚀 Features

✨ Get real-time weather information
🌍 Enter any city name
🌡️ Displays temperature in Celsius
💧 Shows humidity level
⚡ Fast and simple API integration

🛠️ Technologies Used

 🐍 Python
 📡 API (OpenWeatherMap)
 📦 requests library

🔑 Setup API Key

1. Go to OpenWeatherMap website
2. Create a free account
3. Generate API key

 📚 How It Works

🔹 User enters city name
🔹 App sends request to API
🔹 Receives JSON data
🔹 Extracts temperature & humidity
🔹 Displays result on screen

🎯 Project Objective

To understand API integration in Python and fetch real-time data using external services.
